RAMADDA Publish
===============
Allows automated ways of publishing content to a RAMADDA_ server. This document assumes that the user has an account on a RAMADDA_ server.

Installation:
~~~~~~~~~~~~~

``pip install ramadda_publish`` 

To update from previous installation:

``pip install --upgrade ramadda_publish``

Check installation in your favorite command line environment using

``ramadda_publish -h``
