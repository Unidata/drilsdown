IDV_teleport is a command line utility to teleport a IDV Bundle to another space-time. This is often useful when the data for visualization is from multiple data sources or is network intensive for instant visualization. This utility, IDV_teleport then groups vizualization and data at specific events of interest for easy offline exploration inside IDV.
