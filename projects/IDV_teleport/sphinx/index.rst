IDV_teleport
====================
IDV_teleport_ is a command line utility to teleport a IDV_ Bundle_ to another space-time. This is often useful when the data for visualization is from multiple data sources or is network intensive for instant visualization. This utility, IDV_teleport_ then groups vizualization and data at specific events of interest for easy offline exploration inside IDV_.
